# Apa itu RESTful?

**RESTful** adalah istilah untuk implementasi yang mengikuti prinsip-prinsip REST.

Artinya:

- jika hanya memahami konsepnya, itu disebut **REST**,
- jika benar-benar menerapkannya dalam desain API, itu disebut **RESTful**.

## Contoh pemahaman

Kalau sebuah API:

- memakai URI yang jelas,
- memakai method HTTP sesuai fungsinya,
- memisahkan client dan server,
- bersifat stateless,

maka API tersebut bisa disebut **RESTful API**.
