# Perbedaan dokumentasi, spesifikasi, dan definisi API

Walaupun saling berhubungan, dokumentasi, spesifikasi, dan definisi API memiliki tujuan yang berbeda.

| Istilah | Fokus utama | Ditujukan untuk |
|---|---|---|
| Dokumentasi API | Cara memakai API | Manusia |
| Spesifikasi API | Aturan dan perilaku API | Manusia dan sistem |
| Definisi API | Struktur formal yang bisa diproses tool | Mesin |

## Penjelasan singkat

- **Dokumentasi API** membantu developer memahami cara memakai API.
- **Spesifikasi API** menjelaskan aturan dan perilaku API secara lebih formal.
- **Definisi API** mendukung proses otomatisasi oleh tools.

Ketiganya penting karena API yang baik bukan hanya bisa berjalan, tetapi juga harus mudah dipahami, diuji, dan dikembangkan.
