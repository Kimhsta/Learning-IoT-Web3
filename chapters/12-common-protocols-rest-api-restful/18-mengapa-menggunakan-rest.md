# Mengapa menggunakan REST?

REST menjadi sangat populer karena menawarkan cara komunikasi yang sederhana, konsisten, dan mudah diterapkan pada banyak jenis aplikasi.

## Alasan menggunakan REST

- Setiap request dapat membawa informasi yang dibutuhkan.
- REST dibangun di atas HTTP yang sudah mapan di web.
- REST memakai method dasar HTTP yang familiar.
- Semua objek dimanipulasi menggunakan URI.
- Struktur komunikasinya mudah dipahami dan diimplementasikan.

## Contoh pola URI

```bash
/user/object-identifier
```

Dengan pola seperti ini, sistem dapat menyiapkan layanan untuk mengambil resource tertentu berdasarkan identitasnya.
