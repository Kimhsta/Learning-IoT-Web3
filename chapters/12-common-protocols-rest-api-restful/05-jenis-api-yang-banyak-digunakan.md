# Jenis API yang banyak digunakan

Dalam materi ini, ada dua pendekatan yang paling sering dibahas saat membangun layanan antar aplikasi, yaitu:

- **REST** (*Representational State Transfer*)
- **SOAP** (*Simple Object Access Protocol*)

Keduanya sama-sama dipakai untuk komunikasi antar sistem, tetapi cara kerjanya berbeda.

REST biasanya dianggap lebih sederhana, lebih ringan, dan lebih cocok untuk kebutuhan modern seperti web, mobile, dan IoT. Sementara itu, SOAP lebih formal dan cenderung digunakan pada sistem yang membutuhkan aturan sangat ketat.
