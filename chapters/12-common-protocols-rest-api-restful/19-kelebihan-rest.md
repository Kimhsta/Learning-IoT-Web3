# Kelebihan REST

Berikut beberapa keunggulan REST yang paling sering dirasakan dalam praktik:

- Memisahkan antarmuka pengguna dari pengolahan data.
- Membuat sistem lebih modular.
- Memudahkan pengembangan multi-platform.
- Mendukung skalabilitas.
- Memudahkan integrasi dengan layanan lain.
- Cocok untuk public API.
- Lebih ringan dan cepat untuk banyak kasus.
- Kompatibel dengan banyak teknologi seperti PHP, Java, Python, Node.js, dan lainnya.

## Dalam project nyata

Kalau sebuah aplikasi berkembang dari web ke mobile, backend REST biasanya tetap bisa dipakai tanpa perlu dibangun ulang dari nol.
