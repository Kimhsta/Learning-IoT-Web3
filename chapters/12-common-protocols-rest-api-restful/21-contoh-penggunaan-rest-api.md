# Contoh penggunaan REST API

Berikut beberapa contoh pemanfaatan REST API dalam dunia nyata.

## 1. Google Translate dan DeepL

Layanan seperti Google Translate dan DeepL menyediakan API yang bisa dipakai aplikasi lain untuk menerjemahkan teks secara otomatis.

Contoh manfaatnya:

- website multibahasa,
- aplikasi belajar bahasa,
- sistem customer support internasional.

## 2. Wikipedia

Wikipedia menyediakan API yang dapat digunakan untuk mengambil data artikel, pencarian, dan informasi lain secara otomatis.

Ini berguna untuk:

- aplikasi edukasi,
- sistem pencarian informasi,
- monitoring topik atau merek tertentu,
- integrasi konten pengetahuan.
