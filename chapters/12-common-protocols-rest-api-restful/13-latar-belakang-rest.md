# Latar belakang REST

REST diperkenalkan oleh **Roy Fielding**. Ia menjelaskan konsep ini dalam disertasinya pada tahun 2000.

Tokoh ini penting karena juga memiliki kontribusi besar pada perkembangan HTTP. Itulah sebabnya REST sangat berpengaruh dalam cara kita merancang layanan web modern.

## Kenapa sejarah ini penting?

Karena dari sini kita paham bahwa REST bukan sekadar tren, tetapi lahir dari pemikiran arsitektur web yang memang dirancang untuk sistem berskala besar.
