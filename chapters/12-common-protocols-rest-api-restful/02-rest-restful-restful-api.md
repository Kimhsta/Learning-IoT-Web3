# REST, RESTful, dan RESTful API

Tiga istilah ini sering dipakai bersamaan, tetapi artinya berbeda.

- **API** adalah antarmuka agar dua aplikasi bisa saling berkomunikasi.
- **REST** adalah gaya arsitektur untuk merancang sistem komunikasi berbasis web.
- **RESTful** adalah istilah untuk sistem atau API yang mengikuti prinsip REST.
- **RESTful API** adalah API yang dibuat dengan pendekatan REST.

## Ringkasnya

Bayangkan seperti ini:

- **API** = pintu atau penghubung komunikasi.
- **REST** = aturan atau gaya desain pintu tersebut.
- **RESTful API** = pintu yang benar-benar dibuat mengikuti aturan REST.

Jadi, **REST bukan API**, tetapi REST adalah salah satu cara untuk mendesain API.
