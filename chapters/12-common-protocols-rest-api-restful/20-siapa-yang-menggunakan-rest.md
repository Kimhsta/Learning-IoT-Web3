# Siapa yang menggunakan REST?

REST digunakan sangat luas, mulai dari startup kecil sampai perusahaan teknologi besar. Hal ini karena REST cocok untuk membangun layanan yang fleksibel dan mudah diintegrasikan.

Dalam praktik modern, REST sering dipakai untuk:

- public API,
- backend aplikasi mobile,
- dashboard admin,
- integrasi antar layanan internal,
- microservices,
- sistem IoT.

Banyak perusahaan besar maupun layanan digital modern menggunakan pendekatan REST untuk memisahkan business logic dari frontend dan mempermudah integrasi antar sistem.
