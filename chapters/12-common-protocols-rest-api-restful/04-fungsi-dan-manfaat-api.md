# Fungsi dan manfaat API

API penting karena menjadi jembatan antara berbagai sistem. Dengan API, sebuah backend bisa melayani banyak jenis client sekaligus.

## Manfaat API

- Backend bisa dipakai oleh web, mobile, desktop, dan IoT secara bersamaan.
- Aplikasi dengan bahasa pemrograman berbeda tetap bisa saling berkomunikasi.
- Pengembang tidak perlu memahami seluruh isi sistem lain untuk memakainya.
- Sistem menjadi lebih modular dan tidak saling bergantung terlalu ketat.
- Pengembangan frontend dan backend bisa dilakukan secara terpisah.

## Contoh

Satu backend toko online bisa dipakai oleh:

- website pelanggan,
- aplikasi Android,
- aplikasi kasir,
- dashboard admin.

Semua memakai data yang sama melalui API yang sama.
