# Apa itu API?

**API (Application Programming Interface)** adalah sekumpulan aturan dan mekanisme yang memungkinkan satu aplikasi berkomunikasi dengan aplikasi lain.

API membantu aplikasi:

- meminta data,
- mengirim data,
- menggunakan fitur dari sistem lain,
- terhubung tanpa harus tahu cara kerja internal sistem tersebut.

## Analogi sederhana

API bisa diibaratkan seperti **pelayan di restoran**:

- pelanggan memesan makanan,
- pelayan menyampaikan pesanan ke dapur,
- dapur menyiapkan makanan,
- pelayan mengantarkan hasilnya kembali ke pelanggan.

Dalam dunia software:

- **client** = pelanggan,
- **API** = pelayan,
- **server/backend** = dapur.

Client tidak perlu tahu bagaimana server bekerja di dalam. Client cukup tahu cara meminta layanan melalui API.
