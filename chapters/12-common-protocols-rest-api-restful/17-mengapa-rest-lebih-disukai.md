# Mengapa REST lebih disukai?

REST populer karena sederhana dan sesuai dengan kebutuhan pengembangan web modern.

## Alasannya

- Mudah dipahami.
- Memanfaatkan HTTP yang sudah umum.
- Cocok untuk aplikasi web dan mobile.
- Ringan saat bertukar data.
- Fleksibel untuk banyak bahasa pemrograman.
- Mudah dipadukan dengan arsitektur microservices.
- Memudahkan pemisahan frontend dan backend.

## Dampaknya bagi developer

Dengan REST, tim frontend dan backend bisa bekerja lebih mandiri. Ini sangat membantu dalam project besar atau project kolaboratif.
