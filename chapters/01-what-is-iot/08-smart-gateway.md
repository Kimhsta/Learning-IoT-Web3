# Smart Gateway

## Smart gateways

Dalam banyak jenis aplikasi IoT, data mengalir dari perangkat ke gateway. Sebagai contoh, saat data sensor tiba dan ditransfer ke cloud, intelligent gateway dapat mengagregasi data dan bertindak berdasarkan data dari sensor tertentu, sehingga membantu mengoptimalkan komunikasi dengan cloud.

Kecerdasan ini sangat bernilai terutama terkait backhaul seluler (yaitu data plan dan biaya operator). Gateway dengan bahasa pemrograman yang dapat diprogram, seperti Python, memungkinkan pengguna menambahkan kecerdasan pada gateway untuk mengelola data dengan lebih efisien. Alih-alih hanya meneruskan data melalui gateway menggunakan protokol backhaul default, Anda dapat mengoptimalkan solusi Anda menggunakan kode Python. Hal ini dapat membantu Anda menghindari pengiriman terlalu banyak data, menurunkan laju data seluler, dan bertindak cepat ketika ambang batas diatur untuk memberi tahu Anda tentang kejadian tertentu.

Opsi komunikasi backhaul lain pada gateway, seperti Wi-Fi dan Ethernet, juga sangat berguna untuk aplikasi IoT. Gateway dapat dihubungkan untuk menghindari biaya ponsel ketika jaringan lokal tersedia. Namun, ketika hanya cakupan seluler yang tersedia, memiliki opsi untuk berbagai operator nasional dan internasional dapat sangat memudahkan ketika deployment global dibutuhkan.

Tools manajemen gateway yang sudah di-deploy untuk troubleshooting dan diagnosis, serta kemampuan pembaruan firmware jarak jauh, merupakan keharusan untuk aplikasi IoT.
