# Cara Kerja dan Komponen IoT

## Bagaimana Internet of Things bekerja?

IoT mengacu pada perangkat fisik yang menerima dan mentransfer data melalui jaringan nirkabel dengan sedikit campur tangan manusia. Hal ini dimungkinkan karena perangkat komputasi diintegrasikan ke dalam berbagai jenis objek.

## Perangkat pintar

Perangkat ini, seperti televisi, kamera keamanan, atau peralatan olahraga, telah diberi kemampuan komputasi. Perangkat tersebut mengumpulkan data dari lingkungan, input pengguna, atau pola penggunaan, lalu mengomunikasikannya melalui internet ke dan dari aplikasi IoT Anda.

## Aplikasi IoT

Aplikasi IoT adalah sekumpulan layanan dan perangkat lunak yang mengintegrasikan data dari berbagai perangkat IoT. Aplikasi ini menggunakan teknologi Machine Learning (ML) atau Artificial Intelligence (AI) untuk menganalisis data tersebut dan membuat keputusan yang tepat. Keputusan ini kemudian dikomunikasikan ke perangkat IoT, yang akan merespons input secara cerdas.

## Antarmuka pengguna grafis

Antarmuka pengguna grafis dapat digunakan untuk mengelola satu perangkat IoT atau sekumpulan perangkat. Beberapa contoh umum adalah aplikasi seluler atau situs web yang dapat mendaftarkan dan mengontrol perangkat pintar.

Sebagai contoh, smart thermostat (biasanya sesuatu yang cerdas berarti IoT) dapat menerima data lokasi dari smart car. Keterhubungan antarperangkat ini memungkinkan Anda menyesuaikan suhu rumah bahkan ketika Anda tidak berada di sana.

Sistem IoT tradisional bekerja dengan terus-menerus mengirim, menerima, dan menganalisis data dalam sebuah feedback loop. Bergantung pada jenis teknologi IoT, manusia atau sistem AI/ML dapat menganalisis data ini hampir seketika atau seiring waktu.

Sebagai contoh, untuk menentukan kapan waktu yang ideal mengecek thermostat sebelum pulang ke rumah, sistem IoT dapat terhubung ke Google Maps API dan memperoleh informasi terkini tentang lalu lintas di area tersebut. Selain itu, sistem juga dapat menggunakan data jangka panjang mobil untuk memahami kebiasaan berkendara Anda. Lebih lanjut, perusahaan utilitas dapat menganalisis data IoT dari pelanggan yang memiliki smart thermostat untuk mengoptimalkan sistem secara keseluruhan.
