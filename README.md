# IoT E-Book

E-book open source untuk belajar **Internet of Things (IoT)** dari dasar sampai topik jaringan, protokol, arsitektur, dan implementasi modern.

## Deskripsi

Repositori ini berisi materi pembelajaran IoT yang disusun bertahap dalam format Markdown agar mudah dibaca, dipelajari, dan dipublikasikan melalui GitHub.

Materi dibagi per folder agar setiap topik bisa dikembangkan lebih panjang, lebih rapi, dan mudah diberi gambar, diagram, maupun contoh implementasi.

## Tujuan

E-book ini dibuat untuk:
- Mempelajari konsep dasar IoT secara terstruktur.
- Memahami perangkat, sensor, arsitektur, dan konektivitas IoT.
- Mengenal protokol umum dan protokol industri pada IoT.
- Menyediakan bahan belajar yang terbuka dan mudah diakses banyak orang.

## Cara Membaca

Mulailah dari materi pertama, lalu lanjutkan sesuai urutan bab. Setiap folder materi memiliki `README.md` sendiri sebagai isi utama bab, dan dapat dilengkapi gambar pada folder `images/`.

## Pengembangan

Repositori ini masih dalam tahap pengembangan bertahap. Materi akan terus diperbarui, dirapikan, dan dilengkapi dengan ilustrasi agar lebih mudah dipahami.
